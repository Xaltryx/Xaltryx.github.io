// 02 — projects

Things I've Built

001 v2.0

CSV Cleaner CLI

Production-grade Python CLI for automated CSV data quality. Two variants: a zero-dependency stdlib build and a vectorized pandas engine — same interface, same output, built for scale. Strips whitespace, flags bad values (ERROR, NULL, N/A), catches missing fields by column name, and removes duplicate IDs — always producing a clean file and a flagged file with exact error reasons per row.

v2.0 — What's new
N/A added to bad value detection
Pandas variant with full feature parity
Duplicate errors now include the duplicate value
7 pytest tests across both variants
